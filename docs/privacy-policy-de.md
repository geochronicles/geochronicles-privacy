# Datenschutz

[English Version / Englische Version](privacy-policy.md)

**Zuletzt aktualisiert: 2026-01-24**

## Keine Datenerhebung

GeoChronicles ist ein privates Hobbyprojekt. Diese App erhebt, speichert oder überträgt keine persönlichen Daten an eigene Server. Es gibt kein Benutzerkonto, keine Anmeldung und keinen eigenen Backend-Server. Für die Kartendarstellung werden externe Server von CARTO und OpenStreetMap genutzt (siehe unten).

## Standortdaten

Die App greift auf den Standort Ihres Geräts ausschließlich für die Geocaching-Funktionalität zu:

- Ihr Standort wird nur verwendet, um Ihre Position auf der Karte anzuzeigen und nahegelegene Geocaches zu erkennen.
- Standortdaten verbleiben vollständig auf Ihrem Gerät und werden nirgendwohin übertragen.
- Sie können die Standortberechtigungen jederzeit in Ihren Geräteeinstellungen widerrufen.
- Bei der Online-Kartennutzung werden die Koordinaten des angezeigten Kartenausschnitts an die Server von CARTO übertragen, um Kartenkacheln zu laden. Wie bei jeder Internetanfrage wird dabei auch die IP-Adresse Ihres Geräts übermittelt.

## Nur lokale Speicherung

Alle App-Daten (gefundene Geocaches, Einstellungen, Offline-Karten) werden nur lokal auf Ihrem Gerät gespeichert. Es gibt keine eigenen Server zur Datenspeicherung, keine Cloud-Synchronisation und keine externe Speicherung Ihrer App-Daten.

## Kartenkacheln und Kartendaten (Automatische externe Verbindungen)

Die einzige automatische externe Verbindung dieser App ist das Herunterladen von Kartenbildern von CARTO (basierend auf OpenStreetMap). Bei diesen Anfragen werden folgende Daten übertragen: Die öffentliche IP-Adresse Ihrer Internetverbindung sowie Koordinaten (Längen- und Breitengrad) des angezeigten Kartenausschnitts. Wenn Sie die Karte auf Ihren Standort zentrieren, können diese Koordinaten Ihren ungefähren Aufenthaltsort preisgeben. Es werden keine weiteren Daten übertragen und keine Cookies gesetzt. Nach dem einmaligen Herunterladen der Offline-Karten können Sie die Kartenfunktion ohne weitere Netzwerkverbindungen nutzen. Hinweis: Externe Links in der App (z.B. zu Lizenz-, Datenschutzseiten oder Quellenangaben in Geocache-Beschreibungen) öffnen Ihren Browser und erfordern eine Internetverbindung.

Die Datenschutzerklärung von CARTO finden Sie unter: [https://carto.com/privacy](https://carto.com/privacy)

## Ihre Kontrolle

Sie haben die volle Kontrolle über Ihre Daten. Löschen Sie alle App-Daten durch Leeren des App-Speichers oder Deinstallation. Nutzen Sie Offline-Karten, um zu verhindern, dass standortbezogene Daten an externe Server übertragen werden. Nach dem einmaligen Herunterladen der Offline-Karten können Sie die App vollständig offline ohne Netzwerkverbindungen nutzen. Hinweis: Externe Links in der App (z.B. zu Quellenangaben in Geocache-Beschreibungen) erfordern weiterhin eine Internetverbindung.

## Über dieses Projekt

Dies ist ein privates Hobbyprojekt zur Erkundung der Geschichte Wiens durch Geocaching.
